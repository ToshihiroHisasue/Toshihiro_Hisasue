SQLの練習問題

ここではSQLの練習問題を紹介します。

SQLテーブルからすべての列を取得する

SELECT ___ FROM SQL;
解答: *

SQLテーブルから名前と年齢を取得する

___ name, age ___ SQL;
解答: SELECT, FROM

SQLテーブルから年齢が30以上の社員を取得する

SELECT * FROM SQL ___ age >= 30;
解答: WHERE

SQLテーブルから給与の降順（昇順）でデータを並べ替える

SELECT * FROM SQL ___ ___ salary ___;
解答: ORDER BY , DESC(ASC)

SQLテーブルの給与の合計を取得する

SELECT ___(salary) FROM SQL;
解答: SUM(salary)

Sテーブルを作成し、カラムとしてid（整数）、name（文字列）、age（整数）を追加する

___ TABLE S (id ___, name ___, age ___);
解答: CREATE, INT, VARCHAR(50), INT);

Sテーブルに新しい行（id: 1, name: 'Alice', age: 20）を挿入する

___ ___ S (id, name, age) ___ (1, 'Alice', 20);
解答: INSERT INTO, VALUES;

Sテーブルのname列をfull_nameに変更する

___ TABLE S ___ COLUMN name ___ full_name;
解答: ALTER, RENAME, TO;

Sテーブルからageが20未満の行を削除する

___ ___ S WHERE age < 20;
解答: DELETE FROM;

Sテーブルを削除する

___ TABLE S;
解答: DROP;

空のテーブル 
-- 1. 新しいテーブル S を作成
CREATE TABLE S (
    id INT,
    name VARCHAR(50),
    age INT,
    salary INT
);

-- 2. 新しい行を挿入
INSERT INTO S (id, name, age, salary) VALUES (1, 'Alice', 20, 5000);
INSERT INTO S (id, name, age, salary) VALUES (2, 'Bob', 35, 7000);
INSERT INTO S (id, name, age, salary) VALUES (3, 'Charlie', 25, 6000);

-- 3. 名前列を full_name に変更
ALTER TABLE S RENAME COLUMN name TO full_name;

-- 4. 年齢が30以上の行を取得
SELECT * FROM S WHERE age >= 30;

-- 5. 給与の昇順で並び替える
SELECT * FROM S ORDER BY salary ASC;

-- 6. 年齢が20未満の行を削除
DELETE FROM S WHERE age < 20;

-- 7. テーブルを削除
DROP TABLE S;
